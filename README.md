# 原神文件夹管理器

一个用于管理原神游戏文件夹的Android应用程序，支持自动判断当前服务器并在不同服务器之间移动文件。

## 功能特性

1. **服务器自动判断**：根据文件夹大小自动判断当前服务器
   - 大于20GB的文件夹所在的服务器即为当前服务器
   - 显示当前数据服务器（官服或B服）和文件夹大小

2. **智能文件移动**：
   - 从当前服务器文件夹向另一个服务器文件夹移动文件
   - 排除特定文件夹 `unityvulkanpso`
   - 不覆盖原有文件
   - 记录移动日志，便于后续移回

3. **无Root权限运行**：所有功能均采用非Root方式实现

## 支持路径

- 官服路径：`/storage/emulated/0/Android/data/com.mihoyou.yuanshen/file`
- B服路径：`/storage/emulated/0/Android/data/com.mihoyou.ys.bilibili/file`

## 使用方法

1. **检查文件夹大小**：
   - 点击「检查文件夹大小」按钮
   - 程序会自动计算文件夹大小并判断当前服务器
   - 在界面上显示当前服务器和文件夹大小

2. **移动文件**：
   - 当文件夹大小超过20GB时，可以点击移动按钮
   - 移动按钮会显示从哪个服务器移动到哪个服务器
   - 点击按钮后，程序会将当前服务器文件夹中除`unityvulkanpso`外的所有文件移动到另一个服务器

3. **移回文件**：
   - 点击「移回文件」按钮
   - 程序会根据之前的移动日志将文件移回原位置

## 权限说明

- **所有文件访问权限**：需要访问`Android/data`目录
- **存储权限**：读取和写入文件

## 开发环境

- Android SDK 30
- Android Studio
- Java 8

## 构建项目

1. 克隆仓库
2. 使用Android Studio打开项目
3. 连接Android设备或启动模拟器
4. 点击运行按钮构建并安装应用

## 注意事项

1. 首次运行需要授予「所有文件访问权限」
2. 移动文件过程中请勿关闭应用
3. 确保目标文件夹有足够的存储空间
4. 移回文件功能依赖于之前的移动日志
